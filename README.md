#  Featured Projects

### Obsidian Networks
AI-powered machine learning scaffolding platform.

Describe a problem, upload a dataset, and receive a complete trained TensorFlow/Keras model generated by AI.

Features:

• arXiv research integration  
• TensorFlow/Keras API verification  
• automatic architecture selection  
• live training metrics via SSE  
• notebook + model export  
• Docker-sandboxed training environment
